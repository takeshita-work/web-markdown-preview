# 機能一覧

## フォルダ / ファイル選択

- **📂 フォルダを選択** … `showDirectoryPicker()` でネイティブのフォルダ選択。配下の `.md` / `.pdf` を再帰収集してツリー表示（`node_modules` / `.git` は除外）。
- **📄 ファイルを選択（相対パス）** … ヘッダーのタブ左にあるアイコン。クリックでモーダルが開き、ルートからの相対パスを入力して Enter で開く（確定タブ）。フォルダ未選択時は無効。

## ファイルツリー（左サイドバー）

- `.md` / `.pdf` のみ表示。ファイル名の先頭にアイコン（📄 md / 📕 pdf）。
- 起動時は全フォルダ閉じた状態。
- フォルダ展開時、子が**単一のサブフォルダのみ**なら連鎖的に自動展開。
- ファイルを開くと、ツリーがそのファイルまで自動展開・スクロール。

## タブ（仮選択 / 確定選択）

VS Code のプレビュータブ方式。

| 操作 | 動作 |
|---|---|
| シングルクリック | **仮選択**（斜体）。別ファイルを選ぶと置き換わる |
| ダブルクリック（ツリー / タブ） | **確定選択**（pin）。以後消えない |
| 中クリック / × | 閉じる |

- 同名ファイルを複数開くと、タブ名が **`親フォルダ/ファイル名`** 形式になり区別される（ツールチップにフルパス）。
- アクティブタブの背景色はヘッダーと同色で一体化。

## 表示モード（標準 / marp / CSS テーマ）

- ヘッダー下のフッターにある「表示」セレクトで切替。**タブごと**に保持。
- 既定はファイル内容から自動判定：
  - 標準ファイル → 既定 CSS（`markdown-preview` 系を優先）
  - marp ファイル → frontmatter の `theme:`（登録済みなら）／無ければ「marp（frontmatter / 既定）」
- 現在のタブの既定にあたる選択肢には **`(default)`** マーカーが動的に付く。
- 標準フォーマットは**グレー背景＋白いカード**、marp は**スライド中央寄せ・スライド間に余白**で表示。

## ソース表示

- フッターの **`</>`** ボタン、またはプレビュー右クリックメニューで、レンダリング ⇄ ソース表示を切替（**タブごと**に保持）。
- ソース表示でも見出しアウトラインが機能する。

## 見出しアウトライン（右サイドバー）

- 見出し（h1〜h6）を**折りたたみ可能なツリー**で表示。
- クリックで該当見出しへスクロール。
- プレビューのスクロールに追従し、**現在位置の見出しをハイライト**。
- レンダリング / ソース表示の両方で機能。

## 拡大・縮小（ズーム）

- **タブごと × 表示モード（レンダリング / ソース）ごと**に保持（各既定 100%）。
- 操作方法：
  - フッターの **`[−][倍率][＋]`**（一体化 UI）
  - 倍率セレクトで直接選択
  - **Ctrl + ホイール**
  - **Ctrl + Shift + `+` / `-` / `0`**（0 でリセット）
- 段階は切りのいい値（…75/80/90/100/110/125…）。
- 標準は `body` の zoom、marp は SVG スライドの寸法を直接スケール（Chromium で zoom が SVG/foreignObject に効かないため）。

## ライブリロード

- 開いているファイルの更新を 0.5 秒間隔で検知して自動再描画（バックグラウンドタブでは停止）。
- **変更箇所へ自動スクロール**：直前との差分から最初の変更行を特定し、その要素を画面上から約 20% の位置へスクロール。
- **追記・編集時は黄色背景でハイライト**（フェード）。削除（内容が減る変更）ではハイライトしない。

## ディレクトリツリーの自動更新

- ファイルの**追加 / 削除 / リネーム**を反映するため、ツリーを 2.5 秒間隔で再走査（バックグラウンドタブでは停止）。
- **ディレクトリの展開時**・**ファイルのクリック時**にも再走査するので、操作した瞬間に最新の一覧へ更新される。
- 構成に変化が無ければ再描画しない（**ちらつき防止**）。変化時も**展開状態・スクロール位置・選択中ファイル**を保持。
- CSS / marp テーマファイルの増減も再走査時に反映（「表示」セレクトの選択肢が更新される）。

## リンク

- **外部リンク**（`http:` 等）→ ブラウザの別タブ。
- **内部相対リンク**（`.md` / `.pdf`）→ アプリ内タブ（仮選択）で開く。仮選択中のタブからリンクをたどると、元タブは確定に昇格。
- 画像など非対応の内部リンク → blob でブラウザ別タブ。
- 同一文書内アンカー（`#...`）→ 既定どおりスクロール。
- 日本語・空白を含むパスもデコードして解決。

## 印刷 / PDF 出力

- フッターの **🖨**、右クリックメニュー、または **Ctrl + P** で、**コンテンツのみ**を印刷（ブラウザの「PDF に保存」で PDF 化）。
- `@media print` で背景グレーやカードの影を除去してクリーン出力。`print-color-adjust: exact` で選択 CSS の背景色（見出し帯・コード背景）も印刷。
- 標準は左右マージンをコンテンツ padding で確保（印刷ダイアログのマージン設定に依存しない）。marp はスライド 1 枚＝1 ページ（フルブリード）。
- 保存先フォルダはブラウザの仕様上、Web ページからは指定不可（ユーザーがダイアログで選択）。

## コピー（右クリックメニュー）

- ツリー / タブ / プレビュー上の右クリックで：
  - **相対パスをコピー** / **ファイル名をコピー**
- プレビュー右クリック（または挙動）で：
  - **ソースをコピー（全体）**
  - **ソースをコピー（選択範囲）** … レンダリング表示の選択は HTML→Markdown 変換（テーブル・コードフェンス等を復元）、ソース表示の選択は生テキスト。
- コピー成否はトーストで通知。

## サイドバーの開閉・リサイズ

- 左右サイドバーはスプリッタ中央の**ハンドル**で開閉（クリック）・リサイズ（ドラッグ）。
- 閉じるとスプリッタが細いレールとして残り、ハンドルで再展開。
- 幅・開閉状態は `localStorage` に保存。
